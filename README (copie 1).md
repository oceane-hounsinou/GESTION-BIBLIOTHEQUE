# TP 57 : Gestion de Bibliothèque SIL 2021 - Partie 1

## Description
**C'est TP** - Cette épreuve pratique concerne le développement d'une application de gestion de bibliothèque pour les étudiants en SIL (Systèmes Informatiques et Logiciels).

## Objectifs
- Créer une base de données pour la gestion des livres
- Développer des interfaces pour l'ajout et la consultation des ouvrages
- Mettre en place un système de recherche efficace

## Installation
1. Créer la base de données avec `database.sql`
2. Configurer la connexion dans `config.php`
3. Ouvrir `index.php` dans le navigateur
