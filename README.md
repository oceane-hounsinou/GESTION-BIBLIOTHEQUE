# TP 58 : Gestion de Bibliothèque SIL 2021 - Partie 2

## Description
**C'est TP** - Cette épreuve pratique est la suite du développement de l'application de gestion de bibliothèque. Elle ajoute la gestion des emprunts et des retours.

## Nouvelles fonctionnalités
- Système d'emprunts
- Gestion des retours
- Historique des transactions
- Rapports de gestion

## Installation
1. Utiliser la base de données du TP 57
2. Exécuter les scripts d'extension de la base
3. Tester les nouvelles fonctionnalités
